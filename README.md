# SXSW Music 2026 Explorer

A React + TypeScript app for browsing SXSW 2026 music showcases. Built with Vite, Chakra UI, and Motion.

## Demo

[https://mountaindrawn.com/sxsw-2026/](https://mountaindrawn.com/sxsw-2026/)

## Features

- Browse SXSW 2026 music artists and showcases
- View artist photos, descriptions, and venue info
- Listen to sample songs or link out to streaming platforms
- Dark-themed UI with motion animations
- Responsive design

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Testing

```bash
npm test
npm run test:watch
```

## Deployment

Deployed to GitHub Pages via GitHub Actions on push to `main`.
